CLI COMMANDS

Click any command card below to run it inside the live sandbox terminal on the right.

ccshare init
Initialize a new shared configuration, create folders, and select a local port.
ccshare login
Securely link your Claude Pro session cookies to the local rotation pool.
ccshare start
Spin up the local proxy and token rotator server for your coding team.
ccshare status
Check active user load, rotator health, and pool token balances.
ccshare list
List all connected developers and their priority weight allocations.
ccshare clean
Flush session rotation logs and reset cookie caching pools.

Tip:

You can also type custom commands directly into the prompt (e.g. "help", "neofetch", "about", or "motd")!
